Server Name Name of the server that is displayed in the multiplayer tab.
Port Port of the server. Must be unique for all servers. Dont forget to expose the ports you want to use while container startup.
Password visibility visibility_off This password will overwrite any default server password.
Admin Port Admin port of the server (server_admin_port). Must be unique for all servers. Dont forget to expose the ports you want to use while container startup. Please note that OpenTTD will not start a listener on this port when Admin password is not set!
Admin Password visibility visibility_off This admin password (admin_password) will overwrite any default admin server password.
